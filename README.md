# 店小秘主控的 Temu AI 自动上品系统

这是一个以店小秘为主控台、面向 Temu 业务链路的 AI 自动上品系统原型，包含：

- 本地控制台：管理候选商品、查看 AI 计划与任务日志
- AI 服务：生成核价、标题、卖点和执行步骤
- 浏览器插件：注入店小秘和 Temu 页面，读取任务并执行采集、编辑、核验辅助

## 当前进度

第一阶段已开始建设，当前目标是打通最小闭环原型。

- 控制台页面已完成基础界面
- AI 服务已完成 mock 规划 API
- 浏览器插件已完成 MV3 注入骨架
- 页面扫描、调试快照、SKU 匹配原型已完成

## 本地启动

先安装依赖：

```bash
npm install
```

启动 AI 服务：

```bash
npm run dev:server
```

启动控制台：

```bash
npm run dev:dashboard
```

默认地址：

- 控制台：[http://localhost:5173](http://localhost:5173)
- AI 服务：[http://localhost:8787/tasks](http://localhost:8787/tasks)

## 插件加载

当前插件是无需额外打包的原型版本，可直接在 Chrome 或 Edge 中以“加载已解压的扩展程序”方式加载：

```text
apps/extension
```

加载后，在目标站点页面会看到右上角注入的 AI 面板。当前下一阶段将以店小秘页面为主。

## 当前原型能力

1. 展示候选商品任务列表
2. 模拟 AI 核价、标题与上品步骤生成
3. 插件从本地服务拉取当前任务
4. 在页面执行扫描、调试快照上传、SKU 匹配和自动填写尝试
